# Indice degli Esempi Pratici

Questo file contiene un elenco degli esempi Java presenti in questa cartella, relativi all'introduzione ai thread.

1.  **[ES01_ConcorrenzaVsParallelismo.java](ES01_ConcorrenzaVsParallelismo.java)**: Dimostra la differenza tra concorrenza e parallelismo.
2.  **[ES01_CreazioneDiThread.java](ES01_CreazioneDiThread.java)**: Mostra i metodi base per creare thread (estendendo `Thread` e implementando `Runnable`).
3.  **[ES01_MainThread.java](ES01_MainThread.java)**: Illustra il thread principale dell'applicazione.
4.  **[ES01_ProcessoVsThread.java](ES01_ProcessoVsThread.java)**: Spiega la differenza concettuale tra processi e thread.
5.  **[ES02_CreazioneThread.java](ES02_CreazioneThread.java)**: Ulteriore esempio sulla creazione di thread.
6.  **[ES02_ThreadConParametri.java](ES02_ThreadConParametri.java)**: Esempio di passaggio di parametri ai thread.
7.  **[ES02_ThreadCreation.java](ES02_ThreadCreation.java)**: Altro esempio sulla creazione di thread.
8.  **[ES03_CicloDiVitaThread.java](ES03_CicloDiVitaThread.java)**: Dimostra i vari stati del ciclo di vita di un thread.
9.  **[ES03_CicloVitaThread.java](ES03_CicloVitaThread.java)**: Simile a `ES03_CicloDiVitaThread.java`, mostra il ciclo di vita.
10. **[ES04_PrioritaThread.java](ES04_PrioritaThread.java)**: Esempio sull'impostazione e l'effetto delle priorità dei thread.
11. **[ES05_ThreadDaemon.java](ES05_ThreadDaemon.java)**: Illustra l'uso dei thread daemon.
12. **[ES06_ThreadGroup.java](ES06_ThreadGroup.java)**: Mostra come raggruppare i thread utilizzando `ThreadGroup`.
13. **[ES07_InterruzioneThread.java](ES07_InterruzioneThread.java)**: Dimostra come interrompere un thread in esecuzione.
14. **[ES08_UncaughtExceptionHandler.java](ES08_UncaughtExceptionHandler.java)**: Mostra come gestire le eccezioni non catturate nei thread.
